# Agenda de Contactos 

Una aplicación moderna y visualmente atractiva para gestionar contactos, desarrollada con **React** y **Vite**.  
Permite **agregar, visualizar y organizar contactos** con una interfaz fluida y animaciones suaves inspiradas en el estilo “cherry blossom”.

## Tecnologías utilizadas
- **React** – Librería para la interfaz de usuario  
- **Vite** – Bundler ultrarrápido para desarrollo moderno  
- **CSS3** – Diseño adaptable con efectos de vidrio (glassmorphism)  
- **Font Awesome** – Iconografía para mejorar la experiencia visual  

---

## Instalación y ejecución
1. Clona este repositorio:
   ```bash
   git clone https://github.com/KatherineMatosM/Agenda-React
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Ejecuta el entorno de desarrollo:
   ```bash
   npm run dev
   ```

4. Abre en tu navegador:
   ```
   http://localhost:5173
   ```

---

## Componentes principales
- **App.jsx** → Componente raíz que maneja el flujo general.  
- **AgregarContacto.jsx** → Formulario para añadir nuevos contactos (envía datos al API).  
- **ListaContactos.jsx** → Muestra los contactos existentes con opción para ocultar/mostrar.  
- **CherryBlossom.jsx** → Efecto visual animado de pétalos cayendo.  

---

## API utilizada
El proyecto se conecta a la API pública:
```
http://www.raydelto.org/agenda.php
```
Esta API permite realizar operaciones **GET** y **POST** para manejar los contactos.

---

## Estilo visual
El diseño aplica **Glassmorphism** y efectos **gradientes rosados**, con:
- Fondos translúcidos y sombras suaves.  
- Animaciones CSS personalizadas para los elementos en movimiento.  
- Diseño responsive para móviles y escritorio.
